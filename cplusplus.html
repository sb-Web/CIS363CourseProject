An Introduction to Programming in C++

Bjarne Stroustrup began working on the language which would eventually become C++ in 1979. The idea was to introduce classes to a C style language, making one of the first moves into what we know as Object Oriented Programming today. Classes allow sections of code to work in the way people think, by grouping objects together. An example might include a program to store rental car assignments, with a "car" class and a "driver" class to store information for each of those objects. Each class contains the blueprints for an object, including descriptors (attributes) and behaviors (methods).
