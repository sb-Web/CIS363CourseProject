An Introduction to Programming in HTML

The first version of HTML was written in 1993, an accomplishment attributed to Tim Berners-Lee. The initial version of HTML provided a fairly simplistic layout for web pages, but subsequent versions offer many more options. In the current version, HTML5, CSS can be used for formatting while Javascript can be used to add drive user interactions. This very page was written in HTML and CSS.
